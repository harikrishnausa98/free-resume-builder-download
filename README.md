# Free Resume Builder

A modern, client-side React application for building professional resumes with a live split-screen editor.

## 🚀 Features

- **Live Split-Screen Editor**: Type your details on the left, see the resume update instantly on the right.
- **10 Professional Templates**: Choose from Modern, Classic, Minimalist, Creative, Elegant, Tech, and more.
- **Theme Customization**: Pick primary accent colors and professional fonts (Inter, Roboto, Merriweather) to match your personal brand.
- **Drag-and-Drop Reordering**: Easily reorder your skills and experience using a smooth drag-and-drop interface.
- **Rich Text Editing**: Format your professional summary and descriptions with a WYSIWYG editor.
- **100% Privacy**: All data is automatically saved locally in your browser. No server storage means your personal data remains yours.
- **One-Click PDF Export**: High-quality, client-side PDF generation.

## 🛠 Tech Stack

- **Frontend Framework**: React 18 & Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Drag and Drop**: @dnd-kit
- **Rich Text Editor**: react-quill
- **PDF Generation**: html2pdf.js (html2canvas & jsPDF)
